# LIRI Bot App

API's utilized: Spotify, BandsInTown, OMDB

Dependencies: axios, dotenv, node-spotify-api

LIRI a copy of Apple's Siri

LIRI will search for concerts, movies, and song information with simple commands:

    `concert-this`

    `spotify-this-song`

    `movie-this`

    `do-what-it-says`
